- 유령 추적 직업 색상 변경
게임 정산:
- 복기 정보(직업, 임무 진행/킬 수, 사망 원인&살인범)
- 자동으로 대기실로 돌아가기
전체-ショートカット:
- F10: 게임 루트 디렉토리 열기
- F11: 해상도 변경
- Ctrl+F1: 로그 출력
- Ctrl+F2: 게임 중 로그 출력 여부 전환(방장만)
홈-ショートカット:
- Tab: UI 숨기기
채팅-ショートカット:
- Ctrl+C: 현재 채팅창 텍스트 복사
- Ctrl+V: 텍스트를 채팅창에 붙여넣기
- Ctrl+X: 현재 채팅 텍스트 잘라내기
- ↑: 자신의 이전 채팅 기록 추적
- ↓: 자신의 다음 채팅 기록 추적
온라인 검색-ショートカット:
- 왼쪽 Shift: 이전에 가입한 방에 가입
- 오른쪽 Shift: 클립보드의 방에 가입
게임 중-ショートカット:
- F1: 직업 소개 및 스탠드 표시
대기실-ショートカット:
- Shift: 시작 게임 카운트다운 건너뛰기 직접 시작 게임(방장만)
- C: 시작 게임 카운트다운 재설정(방장만)
